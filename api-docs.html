SearhJob API - Документация

Общая информация

Базовый URL API: http://89.35.130.223:25063

Все запросы к API возвращают данные в формате JSON. Для методов, требующих авторизацию, необходимо передавать токен через заголовок Authorization: Bearer {token}.

Для получения токена используйте эндпоинт /api/v1/auth/login.

Доступные эндпоинты

GET /api/v1/jobs

Получение списка всех вакансий.

Параметры запроса
Параметр Тип Описание
page integer Номер страницы (опционально, по умолчанию 1)
limit integer Количество записей на странице (опционально, по умолчанию 10)
Пример ответа
{
  "data": [
    {
      "id": 1,
      "title": "Frontend Developer",
      "description": "We are looking for a Frontend Developer...",
      "company_id": 1,
      "company_name": "Tech Solutions",
      "location": "Moscow",
      "salary": "100000-150000",
      "created_at": "2025-05-01 10:00:00"
    },
    {
      "id": 2,
      "title": "Backend Developer",
      "description": "Experienced Backend Developer required...",
      "company_id": 2,
      "company_name": "Digital Minds",
      "location": "Saint Petersburg",
      "salary": "120000-170000",
      "created_at": "2025-05-02 11:30:00"
    }
  ],
  "meta": {
    "total": 45,
    "per_page": 10,
    "current_page": 1,
    "last_page": 5
  }
}

GET /api/v1/jobs/{id}

Получение информации о конкретной вакансии.

Параметры запроса
Параметр Тип Описание
id integer Идентификатор вакансии
Пример ответа
{
  "data": {
    "id": 1,
    "title": "Frontend Developer",
    "description": "We are looking for a Frontend Developer...",
    "requirements": "- 3+ years experience with JavaScript\n- Strong knowledge of React\n- CSS and HTML skills",
    "company_id": 1,
    "company_name": "Tech Solutions",
    "company_logo": "https://example.com/logos/techsolutions.png",
    "location": "Moscow",
    "salary": "100000-150000",
    "contact_email": "hr@techsolutions.com",
    "created_at": "2025-05-01 10:00:00"
  }
}

GET /api/v1/companies

Получение списка всех компаний.

Параметры запроса
Параметр Тип Описание
page integer Номер страницы (опционально, по умолчанию 1)
limit integer Количество записей на странице (опционально, по умолчанию 10)

POST /api/v1/auth/login

Авторизация пользователя и получение токена.

Тело запроса
{
  "email": "user@example.com",
  "password": "password123"
}
Пример ответа
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "user@example.com"
  }
}

GET /api/v1/auth/user

Получение информации о текущем пользователе.

Требуется авторизация

Пример ответа
{
  "data": {
    "id": 1,
    "name": "John Doe",
    "email": "user@example.com",
    "created_at": "2025-04-15 08:30:00"
  }
}

Коды ошибок

Код Описание
400 Некорректный запрос
401 Не авторизован
403 Доступ запрещен
404 Ресурс не найден
500 Внутренняя ошибка сервера
