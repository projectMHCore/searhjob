Решение проблем с CORS

Если вы видите ошибку Failed to fetch или CORS error, используйте один из следующих методов:

Метод 1: Использование встроенного CORS-прокси

Выберите один из доступных CORS-прокси в инструментах тестирования:

  • CORS Proxy IO
  • All Origins
  • CORS Anywhere

Метод 2: Расширения для браузера

Установите одно из расширений для отключения CORS:

  • Chrome: CORS Unblock
  • Firefox: CORS Everywhere

Метод 3: Использование инструментов командной строки

Используйте curl или Postman для тестирования API без CORS ограничений:

curl -X GET http://89.35.130.223:25063/jobs -H "Accept: application/json"
